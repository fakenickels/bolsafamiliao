Metodologia e princípios

🔍 Metodologia e princípios

1. Dados públicos: Tudo vem do Portal da Transparência, IBGE, CNJ, SIAPE, e outros portais do próprio governo.
2. Verificável: Cada página inclui links diretos para os dados brutos e código para reproduzir as análises.
3. Sem partido: Criticamos políticas de todos os governos. O problema é estrutural, não partidário.
4. Pesquisa aberta: Prove-nos errados. Se um número estiver errado, corrigimos. Verdade > narrativa.
